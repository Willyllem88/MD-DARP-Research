# Cordeau Instances

This directory contains the original Cordeau DARP instances in `.txt` format, as well as the converted JSON files generated by the `instance-converter` script. The JSON files are used as input for the DARPMD solver.

To automatically convert the original `.txt` files to `.json`, you can run the `install_cordeau_instances.sh` script, which will process all `.txt` files in this directory and generate corresponding `.json` files. To run it, simply execute the following command from the root of the project:

```bash
cd build
cmake ..
cmake --build . --target install_cordeau_instances
```
