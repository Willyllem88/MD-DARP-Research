# Python Applications

This directory contains Python applications related to the Multi-Depot Dial-a-Ride Problem (MD-DARP). These applications are designed to complement the C++ solvers by providing tools for instance generation, result visualization, and instance conversion. Each application serves a specific purpose in the workflow of solving MD-DARP instances, making it easier for users to interact with the solvers and analyze their results.

To get started with any of the Python applications, you can install the required dependencies using the following command:

```bash
pip install -r requirements.txt
```
