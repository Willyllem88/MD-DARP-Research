# MD-DARP Launcher

This directory contains a Python application to launch the solvers with a user-friendly interface. This application allows users to easily set up and solve MD-DARP instances by providing a simple interface to specify the parameters and options for the solvers. Users can easily create instances and visualize the results of the solvers using this application.

To run this application just execute the following command from this directory:

```bash
pip install -r ../requirements.txt
python launcher.py
```
